Um brute force que estou desenvolvendo apenas por hobbie e curiosidade, estou compatilhando para aqueles que tem curiosidade e gostaria de melhorar e explorar o codigo
o sistema inicialmente tem o objetivo de usar a rede do tor para ser irrastreavel não que torne impossivel mas que dificulta muito para te encontrar e o sistema troca o ip automaticamente
Fica armazenasdos duas wordlist, uma com usuario e outra com senhas, o sistema pega todos os usuario e testa uma senha para todos, e da um dalay de 3  horas para tentar outra senha ( voce pode mudar o dalay caso queira)
pode ser testado em sistemas que não tem segurança de dois fatores, captcha e tentativas, como exemplo o google workspace
